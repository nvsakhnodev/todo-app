# 📝 Todo App

Простое и аккуратное приложение для управления задачами, созданное с использованием HTML, CSS и JavaScript.

---

##  Demo

 [Ссылка на демо-версию](https://nvsakhnodev.github.io/todo-app/)

---

##  Возможности
- ➕ Добавление задач
- ✅ Отметка выполненных задач
- ❌ Удаление задач
- 💾 Сохранение в localStorage

---

## Технологии
- HTML
- CSS
- JavaScript

---

##  О проекте
Проект создан в рамках изучения frontend-разработки.  
В нём реализована работа с DOM, событиями и хранением данных в браузере.
